Predicting movie ratings based on movie metadata from online movie resources like IMDb, Rotten Tomatoes and Open Movie db.

Feature Identification:
This task will involve identifying features of movies, actors and directors that directly correlate with the movie score. 
Feature Selection:
Dimensionality reduction and regularization techniques can be used to weight features from the feature set to improve the models performance and highly impact the rating of the movie.
Model Building:
Come up with a regression model that predicts the score of any new movie based on features used to build the model.
